Get the latest comics from the web

## Usage
- `!webcomic [comic]`
- `!wc [comic]`

If the comic name is left out, architus will send a message listing all of the supported comics.

## Example
```
!wc xkcd
```

## Supported Comics
- [xkcd](https://xkcd.com)
- [smbc](https://smbc-comics.com)
